HOW IT WORKS

Connect. Normalize. Dispatch. Verify.

Connect

Install the kWh Electric gateway at site. Standard protocols: IEEE 2030.5, OpenADR, SunSpec. No vendor lock-in, no closed OEM cloud.

Normalize

Translates every device language into one clean, consistent data stream. No more per-OEM integration work.

Dispatch

Send charge, discharge, curtail, and shift commands from any platform, in real time — to any DER, at any scale.

Verify

Every dispatch event logged before, during, and after. Provable response, claimable revenue for every asset.
