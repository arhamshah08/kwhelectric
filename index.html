How the gateway works for you. Select a step.

1
Connect
Install the gateway at your site. It connects directly to your batteries, solar inverters, EV chargers, and meters using standard protocols — no vendor lock-in, no closed OEM cloud needed.
2
Normalize
Every device speaks a different language. The gateway translates them all into one clean, consistent data stream — so you see everything in one place, in real time.
3
Dispatch
Send charge, discharge, curtail, or shift commands to your assets from any platform. The gateway handles the translation — safely, in real time, without custom integrations.
4
Verify
Every dispatch event is logged before, during, and after. Your assets' performance is recorded and verifiable — so you can prove response, claim revenue, and build trust.
